ascii => 编排了128个文字符号。只需要7个0和1就可以表示了。

01111111 => 1byte => 8bit

ANSI => 一套标准，每个字符 16bit,2byte

00000000 01111111

中国：gb2312编码，gbk编码 (windows，默认编码),

日本：JIS编码

国家之间编码方式不同，一样的数字有不一样的文字

Unicode: 万国码。

文字量太大，从UCS-2 2个字节，扩充到了UCS-4 4个字节

utf:是可变长度的unicode。可以进行数据的传输和存储。

utf-8: 最短的字节长度8

​ 英文： 8bit，1byte

​ 欧洲文字：16bit，2byte

​ 中文：24bit，3byte

utf-16：最短的字节长度16

总结：

  1. ascii:8bit,1byte

  2. gbk:16bit,2byte windows默认

  3. unicode:32bit,4byte(没法用，只是一个标准)

  4. utf-8: mac默认

    ​ 英文： 8bit，1byte

    ​ 欧洲：16bit，2byte

    ​ 中文：24bit，3byte

gbk和utf-8不能直接就进行转化

gbk <=> 文字 <=> utf-8

在CSS中,可以使用box-shadow给元素边框添加阴影效果:

参数说明:

  • h-shadow: 水平阴影长度,正值向右延伸,负值向左
  • v-shadow: 垂直阴影长度,正值向下,负值向上
  • blur: 阴影模糊距离,值越大阴影越模糊
  • spread: 阴影扩散距离,正值扩散阴影,负值收缩
  • color: 阴影颜色

例如:

  • 0px 5px 5px: 水平0,垂直5px,模糊5px
  • rgba(0,0,0,0.2): 阴影颜色为灰色半透明
