常量

所有字母大写表示常量

可以把所有的变量的字母变成大写就可以认为是常量

Python上没有绝对意义上的常量

变量

变量:可以发生改变的一个量，变量是用来区分不同数据的，可以指向一个内存空间

变量的命名规范：

  1. 必须由字母或数字或下划线_
  2. 不能数字开头，不能纯数字
  3. 不能用python的关键字
  4. 不要用中文（能用，但是不推荐）
  5. 不要太长
  6. 要有意义
  7. 推荐使用下划线命名或驼峰命名法

综上，变量名一定要让人看得懂，看的舒服。

for循环

for循环主要作用于可迭代对象

字符串是迭代的

for循环：

把可迭代的东西中的每一项内容拿出来，挨个的赋值给变量，每一次赋值都要执行一次循环体（代码）

for循环想要计数必须借助range

range(n):从0数到n，不包含n

range(m,n):从m数到n，不包括n

range(m,n,s):从m数到n，不包含n，每次的间隔是s

平时更多使用for循环，while循环是死循环
